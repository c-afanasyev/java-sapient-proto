# java-sapient-proto
Java BSI Flex 335 v2 SAPIENT protobuf descriptors (https://github.com/dstl/SAPIENT-Proto-Files/tree/main/bsi_flex_335_v2_0)

run ```./mvnw protobuf:generate``` to generate java classes
